SI ALGO FALLA
═══════════════════════════════════════════════════════════════════

❌ Build failed
   → Ve a Settings → Environment Variables
   → Verifica OPEN_ROUTER_API_KEY
   → Click "Redeploy"

❌ 404 - Página no encontrada
   → Espera 2-3 min (Vercel aún configura)
   → Recarga la página
   → Si persiste: Redeploy

❌ Assessment tarda mucho
   → ¡NORMAL! Primer attempt tarda ~10 seg
   → Los siguientes son 3-5 seg
   → Es el "cold start" de serverless

❌ API key error
   → Settings → Environment Variables
   → Verifica nombre exacto: OPEN_ROUTER_API_KEY
   → Verifica el valor completo
   → Redeploy
